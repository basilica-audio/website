Download

The latest builds are published on GitHub Releases.

A note on unsigned binaries. These builds are not yet code-signed — the signing and notarization pipeline is still in progress. Your operating system will warn you on first launch:

  • macOS — Gatekeeper will block the first open. Right-click the plugin or app and choose Open, or remove the quarantine flag in Terminal: xattr -dr com.apple.quarantine <file>.
  • Windows — SmartScreen may show “Windows protected your PC”. Choose More info → Run anyway.

Only ever do this for builds downloaded directly from this project’s GitHub releases.
